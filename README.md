# TranslaterHUB - Modern Translation Tool...

![TranslaterHUB](https://images.pexels.com/photos/3861969/pexels-photo-3861969.jpeg?auto=compress&cs=tinysrgb&w=1260&h=750&dpr=1)

TranslaterHUB is a powerful, modern translation platform that breaks language barriers with cutting-edge AI technology. Built with React and powered by advanced language processing capabilities, it offers a seamless translation experience with support for over 100 languages.

## 🌟 Features

- **Real-time Translation**: Get instant translations as you type
- **100+ Languages**: Support for over 100 languages with professional-grade accuracy
- **Smart Detection**: Automatic language detection for seamless translation
- **Text-to-Speech**: Convert text to natural-sounding speech
- **Speech-to-Text**: Convert spoken words to text in multiple languages
- **Keyword Abstraction**: Extract key terms and concepts from text
- **English Quiz**: Interactive quizzes to improve English language skills
- **Grammar Check**: Advanced grammar and style checking
- **Translation History**: Keep track of your previous translations
- **Cross-platform**: Works on desktop and mobile devices

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/TranslaterHUB.git
   ```

2. Navigate to the project directory:
   ```bash
   cd TranslaterHUB
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Create a `.env` file in the root directory and add your API keys:
   ```env
   VITE_GOOGLE_TRANSLATE_API_KEY=your_google_translate_api_key_here
   VITE_FIREBASE_API_KEY=your_firebase_api_key_here
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   VITE_FIREBASE_MEASUREMENT_ID=your_measurement_id
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```
